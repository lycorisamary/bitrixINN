# Bitrix24 Local App: Поиск компаний по ИНН

Локальное PHP-приложение для Bitrix24, которое:

1. Принимает установочные токены через `install.php`
2. Сохраняет их в `auth_data.json`
3. Показывает форму ввода ИНН в `handler.php`
4. Находит компании по ИНН через CRM API
5. Добавляет текущего пользователя в наблюдатели найденных компаний

Портал для тестирования: `https://devcrm.prom23.ru`

---

## 1) Подготовка окружения

### 1.1 Установка PHP
- Установите PHP 7.4+ (рекомендуется PHP 8.x).
- Проверьте установку:

```bash
php -v
```

### 1.2 Установка ngrok
- Установите ngrok с официального сайта: [https://ngrok.com/download](https://ngrok.com/download)
- Зарегистрируйте аккаунт ngrok: [https://dashboard.ngrok.com/signup](https://dashboard.ngrok.com/signup)
- Выполните авторизацию токеном:

```bash
ngrok config add-authtoken <YOUR_NGROK_TOKEN>
```

---

## 2) Настройка приложения

1. Создайте папку проекта и скопируйте файлы:

```text
bitrix-inn-app/
├── config.php
├── install.php
├── handler.php
├── auth_data.json      # создается автоматически после установки
├── .gitignore
└── README.md
```

2. Откройте `config.php` и проверьте настройки:
   - `portal_url` = `https://devcrm.prom23.ru`
   - `local_port` = `8000` (или ваш порт)
   - `auth_file` = путь к `auth_data.json`
   - `api_timeout` = таймаут API-запросов

---

## 3) Запуск локального сервера

Из папки `bitrix-inn-app` запустите:

```bash
php -S localhost:8000
```

После запуска приложение будет доступно локально:
- `http://localhost:8000/install.php`
- `http://localhost:8000/handler.php`

---

## 4) Настройка ngrok

В отдельном терминале запустите:

```bash
ngrok http 8000
```

Скопируйте HTTPS-адрес вида:

```text
https://xxxx-xx-xx-xx-xx.ngrok-free.app
```

---

## 5) Регистрация приложения в Bitrix24

1. В Bitrix24 откройте раздел разработчика (Приложения -> Разработчикам / Developer resources).
2. Создайте новое приложение:
   - Тип: **Локальное приложение**
3. Заполните URL:
   - **Путь обработчика (handler)**:  
     `https://<ваш-ngrok>/handler.php`
   - **Путь установки (install)**:  
     `https://<ваш-ngrok>/install.php`
4. Права доступа:
   - CRM (компании, изменение компаний)
   - Пользователи (для `user.current`)

---

## 6) Установка приложения

1. Нажмите кнопку установки приложения в Bitrix24.
2. Bitrix24 отправит параметры в `install.php`.
3. Проверьте, что появился файл `auth_data.json` в корне проекта.
4. Если файл не появился:
   - проверьте доступ на запись в папку проекта
   - убедитесь, что URL `install.php` указан верно
   - проверьте, что ngrok и локальный PHP-сервер запущены

---

## 7) Использование

1. Откройте приложение из меню Bitrix24.
2. Введите ИНН (10 или 12 цифр).
3. Нажмите кнопку **Найти и привязать**.
4. Приложение:
   - найдет поле ИНН в `crm.company.fields`
   - выполнит поиск в `crm.company.list`
   - получит текущего пользователя через `user.current`
   - добавит пользователя в наблюдатели каждой найденной компании (`crm.company.update` с полем `OBSERVER_IDS`)
5. На странице отобразится:
   - ID текущего пользователя
   - число найденных компаний
   - число успешных/неуспешных привязок
   - детали ошибок (если есть)

---

## Безопасность и важные замечания

- `auth_data.json` добавлен в `.gitignore`.
- Не храните в репозитории чувствительные токены.
- Все входные данные валидируются:
  - ИНН: только 10 или 12 цифр
  - установочные параметры в `install.php`: `AUTH_ID`, `AUTH_SECRET`, `MEMBER_ID`
- Для вывода текста используется `htmlspecialchars`.
- Если `auth_data.json` отсутствует, приложение покажет понятную ошибку.
